# QuranMemorizer (حفظ القرآن)

هذا مشروع جاهز للبناء على GitHub Actions وإخراج APK.

## تشغيل سريع عبر GitHub Actions
1) ارفع هذا المشروع على GitHub
2) أضف ملفات الصفحات (مصحف المدينة) داخل:
   `app/src/main/assets/mushaf/page-001.json ... page-604.json`
3) افتح Actions -> Build APK -> Run workflow
4) حمّل Artifact: app-debug-apk (وفيه app-debug.apk)

> ملاحظة: داخل المشروع يوجد page-001.json مثال للتجربة فقط.
